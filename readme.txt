== Description ==

Adds a healthy does of Vitamin A to the WordPress Admin, putting carrots into all your checkboxes and radio buttons.

== Frequently Asked Questions ==

= Is it a good idea to use this plugin on a site with a lot of users? =
Probably not.

= Is it a good idea to use this plugin on a site with a lot of buns? =
Definitely.

= When can I use this plugin? =
International Carrot Day (April 4), National Eat Your Vegetables Day (June 17, USA only), World Vegetarian Day (October 1), and World Vegan Day (November 1).

= What do you get when you cross a carrot with the block editor? =
Rootenberg.
